RoBotanist is a plant watering robot. It's composed of a Raspberry Pi that is set up as a NodeJS server which controls a watering pump. Users can access the companion app through their browser to select what plant they want to water. The app then uses its custom API to query a database of plants to retrieve data on how often that plant should be watered. This data is then posted as an HTTP request to the Node server on the Pi which remotely configures RoBotanist's watering frequency. There is also a "water now" button for those who prefer manual control.

Check it out on Github!

Gitocracy is a platform that seeks to democratize project development. Intended for open-source or community projects, Gitocracy allows developers to link their project's GitHub repo to the platform. Gitocracy then uses GitHub's API to retrieve data from the repos "issues" and displays the information so that users can see what bugs and features are currently being worked on. Users are then able to suggest features for the product and use an upvote/downvote system to nominate other users' suggestions. Devs can then decide to add suggestions as issues.

Check it out on Github!

Climbr is an app for rock climbers. It sets out to be the first graphically oriented database where climbers can access climbing routes and locations, all submitted by their peers.

Check it out on Github!

Sharebear is a lending application made in React that offers users the ability to register and share items with their neighbors. Guests are able to search by zipcode and browse through a list of available items to borrow. Once a guest registers, they can list items or make arrangements to borrow items from others.

Check it out on Github!
